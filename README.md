# Python Practice Folder

This folder contains a single Jupyter notebook for beginner-level Python practice.

## Folder Contents

- `exercise1.ipynb`: A notebook with 10 Python exercise questions and student-style solutions.

## What Is Inside `exercise1.ipynb`

The notebook includes these topics:

1. String reverse and every 2nd character
2. Squares of even numbers using list comprehension
3. Character frequency count using a dictionary
4. Indices of odd numbers using `enumerate()`
5. Right rotation of a list using slicing
6. Creating a dictionary from two lists using `zip()`
7. Manual iteration using `iter()` and `next()` with `StopIteration` handling
8. Flattening a nested list using list comprehension
9. Palindrome check using slicing
10. Second largest number in a list without sorting

## How To Use

1. Open `exercise1.ipynb` in VS Code or Jupyter Notebook.
2. Run cells from top to bottom.
3. Edit input examples to test different outputs.

## Purpose

This notebook is useful for practicing core Python concepts in a simple, beginner-friendly way.
